// HydroCity Trader location for the Dr Jones Trader Mod

// To Install:
// Make sure you have the "Trader" mod by Dr Jones
// Put this file "hydrocity.txt" in the "Trader" folder on your server.
// This will be in your Profile or SC folder depending on you provider.
// Open your existing "TraderObjects.txt" file.
// At the bottom, on a new line just above "<FileEnd>", add "<OpenFile> hydrocity.txt".
// Or if you are familiar with editing the Trader config files you can paste this
// file's contents into your existing TraderObjects minus the last "<FileEnd>" line.

// Description:
// Completely redesigns the Solnichniy industrial area and adds a massive
// hydropower dam structure with full reservoirs and a huge underground
// bunker complex. Trader people are on the lower floor.
// The car seller is at the front / coast entrance on the road.
// Out to sea there is a huge wind turbine farm and an extended dock with
// a Rify ship.

// There is a super secret underground entrance which will require
// the "Punch Card" item so can have a unique treasure hunt on your server
// by hiding a card somewhere on the map for players to find, and the winner gets
// whatever you hide behind the door.

// This was made steadily over about six months for my server "Essex Reapers"
// It will continue evolve as new ideas come along.

// Extra Features:
// You will need the "Banking" mod for the ATMs to appear
// If you have the "Airdrop Upgraded" mod you will get red smoke on the top of the tower.
// If you have the "OP_BaseItems" mod there will be spotlights at night at the entrances.

// Things that will be changed / improved soon:
// A few gaps where you can see into areas you shouldn't.
// Fire Barrels are slashed out due to issue with 1.21 update.

// Things you can change:
// The water levels can be adjusted, just change the height of any 
// "StaticObj_water_pond_50x50m" items below that share the same height
// Or delete them for empty reservoirs.
// The seperate pools of water are grouped and labeled below

// If you remove the trader people and safe zones this can be used as a
// massive base for players or admins. Add a few locked BBP doors and
// it's a fortress. 

// Marker Cordinates for "VanillaPlusPlusMap" 12904.9, 170, 6260 (Tip of Radio Tower)
